# Decisions

## Purpose

Record important technical decisions so future AI agents do not repeatedly revisit settled choices.

## Decision Template

```markdown
# Decision: <title>

## Date

YYYY-MM-DD

## Context

What problem required a decision?

## Decision

What did we choose?

## Reason

Why was this option selected?

## Alternatives Considered

What else was considered?

## Impact

What changes because of this decision?
```

## Rules

- Record important architecture and research decisions.
- Do not silently reverse prior decisions.
- If a decision changes, document why.
- Keep decisions short and factual.

## Checklist

- [ ] Context recorded
- [ ] Decision clear
- [ ] Reason included
- [ ] Alternatives noted
- [ ] Impact stated
